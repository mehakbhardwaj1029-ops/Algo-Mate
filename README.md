Tech Stack

Installation
 Clone the repository
   git clone https://github.com/yourusername/algo-mate.git
   cd algo-mate

Backend Setup
1. cd backend
2. mvn clean install
3. mvn spring-boot:run

Frontend Setup
1. cd frontend
2. npm install
3. npm start

Navigate to "http://localhost:5173" to access it
